üí° Explanation

This is the introductory level designed to get you familiar with connecting to the Bandit server via SSH. SSH (Secure Shell) is a protocol used to securely connect to remote servers.

  • -p 2220: Specifies the port number (default SSH port is 22, but Bandit uses 2220)
  • The format is username@hostname
  • After successful login, you'll have access to the Bandit server
